    Adding a Built-in Stage Background

    1. In the lower right corner of the Snap! window, select the Stage icon (shown right).
    2. Choose "Backgrounds..." from the File () menu.
    3. Choose a background.
    4. Click "Import."
    5. Click "Cancel" to close the dialog box.

  1. Add more backgrounds if you want.
  2. Change the background to the first background that you want to appear.

    Changing a Stage Background

    1. If it isn't selected already, Select the Stage icon in the lower right corner of the Snap! window.
      2. The "Backgrounds" tab isn't always visible. It replaces the "Costumes" tab when the stage icon is selected.
    2. Click the "Backgrounds" tab above the scripting area.
    3. Select a different background.

Adding a Costume

  1. Change the costume for your first sprite. (You can add more sprites later.)
    Changing a costume is almost like changing a stage background! The only differences are in bold below.

    Adding a Built-in Sprite Costume

    1. In the lower right corner of the Snap! window, select the Sprite icon (shown right).
    2. Choose "Costumes..." from the File () menu.
    3. Choose a costume.
    4. Click "Import."
    5. Click "Cancel" to close the dialog box.

  1. Add more costumes (for this sprite) if you want.
  2. Rename any costumes you want to by right clicking on the costume in the "Costumes" tab and selecting "rename."
  3. Change the costume for this sprite to the first costume that you want it to have.

    Changing a Sprite Costume

    1. Select the Sprite icon if it isn't selected already.
      2. The "Costumes" tab isn't always visible. It replaces the "Backgrounds" tab when a sprite icon is selected.
    2. Click the "Costumes" tab above the scripting area.
    3. Select a different costume.
  1. Fix any issues with your costumes.

Adding More Sprites

  1. If your project needs more than one sprite, add more, and then give them costume(s) too.

    Adding a Sprite

    • Clicking on the button just below the stage.
      The new sprite, automatically named "Sprite(2)", will appear on the stage. The pictures of the sprites beneath the stage let you select the sprite you want to work with.

Naming Your Sprites

  1. If you want, give names to your sprites so you can keep track of them.

    Naming a Sprite

    • Look for the name in a gray box above the scripting area. Click inside the name and you'll be able to edit it.

Use Your Own Images

  1. If you want, add your own images for sprite costumes or stage backgrounds.

    Drawing an Image in Snap!

      You can read more about the drawing tools in the Snap! Reference Manual. Look in the index for "Paint Editor".
    1. Click the "Backgrounds" tab or the "Costumes" tab.
    2. Click the "paint" icon ().
    3. Create your background or costume.
    4. Click "OK."
    If you use an image that someone else created, check to make sure that you are allowed to use it. If you are, leave a comment in your Snap! code acknowledging the creator. You learned about adding comments in Unit 1 Lab 6 Activity 1: Texting Time.

    Importing an Image from Your Computer

    1. Select the stage icon or a sprite icon depending on where you want the image to go.
    2. Drag the image file from your computer anywhere into the Snap! window.
    3. Click the "Backgrounds" or "Costumes" tab, and select the background or costume.
    You may have imported an image from a webcam before on Unit 1 Lab 4 Activity 3: Investigating and Storing Images
