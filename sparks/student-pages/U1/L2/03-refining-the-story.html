In this activity, you'll use operator functions to generate random numbers and to join letters together to make new words to improve the story.

Using a Random Number

Notice that the input slot for numbers is a different shape. In Snap!, the shape of an input slot tells you what kind of input you can put in that slot. A rectangle will accept any kind of text (letters, numbers, punctuation, emojis, etc.), but an oval will only accept a number.

You've used special random blocks that were designed especially for this project to generate random feelings, jobs, actions, places, etc. Programming languages (including Snap!) have a built-in way to generate random numbers.

  1. If your project isn't open already, log in to Snap! and open your "Super Short Stories" project by choosing "Open" from the Snap! File menu ().
  2. Locate in the palette on the left hand side of the Snap! window, drag it into the Scripts area, and click it a few times.
    They could just click on the block in the palette without dragging it out; is there a reason not to teach that? -bh
  3. Change the inputs to pick random number a few times, and each time you do, click it a few times to see what happens.
    Do you want to specify whole number inputs? (Or, alternatively, suggest they also try decimal-fraction inputs?) -bh
  4. Describe what the pick random number block does.
  5. Drag pick random number into the number input slot of Super Short Story, and run that block a few times to verify that a random number is used in the story. If it isn't working right, work with another student to sort out the problem.
