Creating a Snap! Account

  1. Go to https://snap.berkeley.edu/run.
  2. Image should be redone at a higher resolution. --MF, 8/12/21
    Click the Cloud menu (it looks like a cloud: ), select "Signup...", and follow the instructions.

You should be logged in whenever you are in Snap!, and you should log out whenever you are done. You can use the Cloud menu to log in, log out, check if you are logged in, and change your password.

Saving Your Project

  1. Make sure you are logged in to Snap! by checking that your name appears in the Snap! Cloud menu (), and logging in if necessary.
  2. Save this project to your Snap! account by choosing "Save" from the Snap! File menu (it looks like a piece of paper with a corner folded: ).

Be sure to save your projects after an important change and, if you are using a shared computer, whenever you're done for the day.
