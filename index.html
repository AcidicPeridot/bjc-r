Welcome to the Beauty and Joy of Computing course listing. These courses offer similar versions of BJC, but are tweaked for specific audiences.
The CS10 courses will generally reflect newer content which we are trying out in class. Some of these materials may not be quite as polished as the main BJC course and may be faster paced. More info about the CS10 course is available at cs10.berkeley.edu.
The main page for the Beauty and Joy of Computing Project is at bjc.berkeley.edu.
You can also view a "bleeding-edge" version of this website at beautyjoy.github.io/bjc-r. Github also provides an excellent way to submit feedback, bug reports, or ask questions. View the "issues" at github.com/beautyjoy/bjc-r/issues.
