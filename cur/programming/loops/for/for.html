The numbers in the for block work like you probably think they do: the inner script is run enough times to count from the left number (1) to the right number (10).

What's new to you here, though, is the orange oval with "i" in it. By using this i, your inner blocks can know which loop they are on. Here's how:

The block is a counter, and you use it in place of a number you type to stand for the particular loop that it taking place.

Think of the counter like a box with a name—this one has the name i. This box is made for you by the block, and holds a different number each time through the loop. The first time it will hold 1, or whatever is the left number in the for block.

You use it by dragging from the source within the for block down to the slot in which you will use it.
