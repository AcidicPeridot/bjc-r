Let's try making a few more blocks that use if and if-else. We're mostly going to make "reporter" blocks, that actually return a value. These have rounded edges and are often called "functions."

  • Make a block that when given a light color, reports the appropriate action. For example should report "go!"
  • Make a block that takes in a percentage and reports the associated letter grade. For example, should report "C."
  • Make a block that takes in a temperature and returns whether water will be liquid, solid, or gas.
